# observations

this directory contains ground-truth annotations and the outputs generated by the pipeline.

## structure

- `*_ground_truth.rttm` -- original ami annotations used to automatically calculate the diarization error rate (DER).
- `outputs/enhanced_*.wav` -- the cleaned audio from phase 1, which is passed to phase 2.
- `outputs/diarization_*.json` -- pyannote's speaker timestamps.
- `outputs/diarization_*.rttm` -- standard format predictions.

## inspection instructions
to test the enhancement suite independently:
```bash
python -m pytest tests/ -v -s
```

to run the full end-to-end pipeline (enhancement + diarization + deep evaluation):
```bash
python run_pipeline.py --input screening_notebooks/sample_input_and_output_files/EN2002a.wav --output-dir observations/outputs
```

to run automated batch tracking across edge cases (overlap, variable speakers):
```bash
python evaluate_batch.py
```
